# AI 通识教育基础

一本面向非计算机读者、高中生与大学生的免费电子书。

本书从古代逻辑、计算工具、机械计算、图灵机、电子计算机、符号 AI、专家系统、机器学习、深度学习讲到大模型时代，并补充 AI 基本原理、AI 使用素养、事实核查、隐私、版权和治理等内容。

## 适合读者

- 非计算机背景但希望理解 AI 的普通读者
- 高中生、大学生与通识课程学习者
- 希望在课堂中引入 AI 基础内容的教师
- 想了解大模型、AI 使用方法和 AI 风险的学习者
